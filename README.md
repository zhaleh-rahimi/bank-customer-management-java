# Bank customer management system
## This is an app to manage 2 types of customers. natural customers and legal customers in a banking system.
## functionalities are:
- select the type of customer
- create the customer
- search the customer
- edit the customer
- delete the customer

## materials used in this app are:
- java 
- maven
- html
- jsp
- css
- mysql server
- mysql workbench
